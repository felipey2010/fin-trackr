# FinTrackr

A modern and scalable personal finance analysis web application built with Next.js, Tailwind CSS, and shadcn/ui.
The platform helps users track income and expenses, analyze spending behavior, and gain actionable insights to improve financial health.

The application supports secure authentication using NextAuth.js, allowing multiple users to manage and revisit their financial data safely.

## Key Features

- User authentication and secure data access
- Income and expense tracking
- Financial analysis based on user income
- Savings and overspending insights
- Smart spending recommendations
- Interactive financial dashboard with charts
- Responsive and clean user interface
- Scalable and maintainable architecture following best practices

## Purpose

This project aims to provide a simple yet powerful tool for personal financial awareness, helping users understand their spending patterns and make better financial decisions.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- NextAuth.js
- PostgreSQL (or compatible database)
- Prisma / Drizzle ORM
